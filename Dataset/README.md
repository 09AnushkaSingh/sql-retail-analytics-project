## Dataset Information

This folder contains the dataset used for the SQL Retail Analytics project.

### Files Included

- **sample_online_retail_clean.csv**
  - 10% random sample of the fully cleaned dataset
  - Created to comply with GitHub file size limits
  - Retains the same schema as the full dataset

### Original Dataset (Not Uploaded)

- The complete cleaned dataset contains ~397,000 rows
- It is intentionally not uploaded due to GitHub size constraints
- All SQL analysis was performed on the full dataset locally

### Columns Description

- InvoiceNo
- StockCode
- Description
- Quantity
- InvoiceDate
- UnitPrice
- CustomerID
- Country
- TotalAmount
- Year
- Month
