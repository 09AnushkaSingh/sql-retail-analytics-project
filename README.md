# Retail Sales & Customer Analytics (SQL Project)

## Project Overview:
**Project Type:** End to End Retail Data Analytics <br>
**Skills Demonstrated:** 
  * Data Cleaning & Feature Engineering (Python)
  * SQL Analytics (CTEs, Window Functions, Business Logic)
  * Business Insight Generation 

This project analyses real world retail transaction data to understand customer purchasing behavior, sales performance and revenue drivers.

The workflow follows an industry style analytics pipeline:
 * 1.Raw data cleaning and preparation using Python
 * 2.Creation of an analysis ready dataset
 * 3.Business focused analysis using SQL
 * 4.Insight generation for decision making


## Problem Statement & Objectives
Retail business generate massive transaction data but raw data alone does not creat value. <br>
This project aims to answer:
 * How does revenue vary across time, customers and products?
 * Which customers and transactions contribute most to revenue?
 * What sales patterns can help improve pricing and inventory decisions?
 * How can SQL be used to extract business insights from large datasets?

## Key Objectives
 * Clean and prepare raw retail data for analysis
 * Engineer meaningful features for business analysis
 * Perform structured SQL analysis on transactional data
 * Extract actionable insights for business stakeholders
